In the panel to the left are pages for different components used in the project that can be tested individually.

Key information to note:
For MDX files:
```js
import { Meta } from '@storybook/addon-docs'; // Imports the Meta
...

<Meta title="example/Introduction" /> // Sets the 'directory' or tab in the side bar
```

For JSX files:
```js
import { Header } from '../components/Header'; // Import component

export default {
  title: 'Example/Header', // For setting the 'directory' in the side bar
  component: Header, // <- Map it to the component
  parameters: {
    // More on Story layout: https://storybook.js.org/docs/react/configure/story-layout
    layout: 'fullscreen', // For defining layout
  },
};

const Template = (args) => <Header {...args} />; // Instantiating an instance of the component

export const LoggedIn = Template.bind({}); // Cloning a component instance
LoggedIn.args = { // Passing the arguments used for ...args to the component for this instance
  user: {
    name: 'Jane Doe',
  },
};
```
